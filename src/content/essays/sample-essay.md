This file exists so the platform builds and demos without the publication's
content. The Reducing Valve's essays are not part of this repository: the
platform is MIT-licensed code, the essays are CC BY-NC 4.0 prose, and keeping
them apart keeps both licenses honest.

If you are adopting this platform for your own publication, replace this file
with your first essay. Frontmatter takes a title, a dek (the standfirst under
the title), a date, an optional `updated` date, a type of `essay` or `note`,
an optional issue number, and tags.
